A simple personal finance tracker built with PHP, HTML/CSS and JavaScript.

## Overview

This project is designed to help individuals track their income and expenses in a lightweight web-based system. You can enter incomes, expenses, generate reports (including CSV and PDF export), and view summaries of your financial status.

## Features

- Add and delete income records  
- Add and delete expense records  
- View detailed lists of incomes and expenses  
- Generate summary reports (e.g., monthly totals)  
- Export data as CSV or PDF  
- Responsive front-end using JavaScript & CSS  
- MySQL database backend for persistent storage  

## Requirements

- PHP (version 7.x or higher)  
- MySQL / MariaDB  
- Web server (Apache / Nginx)  
- Browser with JavaScript enabled
